# AWS CLI Automation Project

## Project Overview

This project demonstrates AWS infrastructure automation using AWS CLI and Bash scripting.

The project automates EC2 instance management using terminal commands and shell scripts.

---

## Technologies Used

- AWS CLI
- Amazon EC2
- Bash Scripting
- Ubuntu Terminal
- Git & GitHub

---

## Features

- Configure AWS CLI
- Start EC2 instance using CLI
- Stop EC2 instance using CLI
- Automate EC2 operations using Bash script

---

## AWS Services Used

- Amazon EC2
- IAM

---

### Start EC2 Instance

aws ec2 start-instances --instance-ids <instance-id>
